# Database Management

## 1. DB의 논리적 구조

### 1.1 Table

- Excel sheet와 같은 표 형식으로 data 저장

### 1.2 Schema

- Table들을 모아놓은 방
- Schema 관리
    - Schema 목록 조회
        - show schemas;
    - Schema 생성
        - create schema {schema name}
    - Schema 삭제
        - drop shema {schema name}

- 사용자 관리
    - 사용자 생성 / 권한부여
        - grant {권한} on {schema name}.{table name} to {user name}@host identified by '{password}';
    - 비밀번호 변경
    - 사용자 삭제
    - 권한 확인
- DB 접속
    - $mysql -h{host} -P{port} -u{user name} -p
