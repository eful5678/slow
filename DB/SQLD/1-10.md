## 9. DELETE
### 풀이 : 4
```sql
CREATE TABLE T(
    C INTEGER PRIMARY KEY,
    D INTEGER
);

CREATE TABLE S(
    B INTEGER PRIMARY KEY,
    C INTEGER REFERENCES T(C) ON DELETE CASCADE
);

CREATE TABLE R(
    A INTEGER PRIMARY KEY,
    B INTEGER REFERENCES S(B) ON DELETE SET NULL
)

DELETE FROM T;

SELECT *
FROM R
JOIN S
ON R.B = S.B
JOIN T
ON S.C = T.C
```
### 개념정리
- Constraints
- DELETE CASCADE
- DELETE SET NULL
